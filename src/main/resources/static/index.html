🏷️ 多标签过滤 (新功能)

GET /api/snippets?tags=java,spring&logic=or NEW

使用 OR 逻辑过滤：返回包含 java 或 spring 标签的片段

curl "http://localhost:8080/api/snippets?tags=java,spring&logic=or"
GET /api/snippets?tags=java,spring&logic=and NEW

使用 AND 逻辑过滤：返回同时包含 java 和 spring 标签的片段

curl "http://localhost:8080/api/snippets?tags=java,spring&logic=and"

📋 基础端点

GET /api/snippets

获取所有代码片段

GET /api/snippets?tag=java

按单个标签过滤（向后兼容）

GET /api/snippets?language=java

按编程语言过滤

GET /api/snippets?keyword=repository

按关键词搜索（标题/描述）

GET /api/snippets/{id}

获取单个片段详情

GET /api/snippets/tags

获取所有可用标签列表

✍️ 操作端点

POST /api/snippets

创建新的代码片段

curl -X POST http://localhost:8080/api/snippets \ -H "Content-Type: application/json" \ -d '{"title":"My Snippet","code":"System.out.println();","language":"java","tags":"java,example"}'
PUT /api/snippets/{id}

更新现有片段

DELETE /api/snippets/{id}

删除片段
